# agcli ‚Äî LLM/Agent Reference

## Tier 1: Quick Reference Card

### What is agcli?
Rust CLI + SDK for Bittensor. Manage wallets, stake TAO, transfer tokens, register on subnets, set weights, query chain state, monitor subnets. All commands: `--output json|csv`, `--batch` for hard errors, `--yes` to skip prompts.

### Install
```bash
cargo install --git https://github.com/unconst/agcli
```

### Exit Codes
- 0 = success, 1 = error
- Batch/JSON mode: `{"error": true, "message": "..."}` on stderr

### Key Concepts
- **TAO** = native token, 1 TAO = 1e9 RAO (u64)
- **Subnet** = independent network, netuid (u16), 0 = root
- **Coldkey** = encrypted signing key, **Hotkey** = automated plaintext key
- **Weights** = validators score miners, u16 values normalized to [0,1]
- **Dynamic TAO** = each subnet has alpha token with AMM (constant-product pool)
- **Commit-reveal** = privacy weight setting (blake2 hash ‚Üí reveal)
- **Commitment** = miner endpoint publication (replaces Axon)

### Agent / Non-Interactive Mode

Every command is fully scriptable with zero stdin blocking when all required args are provided. All flags use `--flag` syntax (no positional args).

```bash
# Hard-error mode: missing required args = error with hint, never prompt stdin
agcli --batch --output json wallet create --name w1 --password mypass --yes

# Persistent batch mode via config
agcli config set --key batch --value true

# Spending limits for safety
agcli config set --key spending_limit.97 --value 100.0    # Max 100 TAO on SN97
agcli config set --key spending_limit.* --value 500.0     # Global max
```

### Subnets
- `subnet list [--at-block N]` ‚Üí `[{"netuid","name","n","max_n","tempo","emission","burn_rao","owner"}]`; no `--netuid` / no exit **12** for unknown SN; e2e `subnet_list` calls `list_subnets` (`pin_latest` + `get_all_subnets_at_block` + `get_all_dynamic_info_at_block`)
- `subnet show --netuid N [--at-block N]` ‚Üí full details + Dynamic TAO pricing
- `subnet metagraph --netuid N [--uid UID] [--full] [--save] [--at-block N]`
- `subnet hyperparams --netuid N [--at-block B]` ‚Üí all hyperparameters
- `subnet register` ‚Äî create new subnet (burn lock cost); no `--netuid` / no exit **12**; lock amount via `subnet create-cost`; e2e **`subnet_register_plain`** reuses same `get_subnet_registration_cost` log as **`subnet_create_cost`**
- `subnet register-with-identity --name ...` ‚Äî register + subnet identity in one call; optional `--github` / `--url` / etc.; pre-unlock validation like `identity set-subnet`; no `--netuid`; e2e `subnet_register_with_identity` uses `get_subnet_identity`
- `subnet create-cost` ‚Äî read-only subnet creation lock (TAO) for `register` / `register-leased`; no `--netuid`; RPC/runtime API read errors only
- `subnet register-leased [--end-block N]` ‚Äî create leased subnet; no unknown-netuid preflight; lock amount via `subnet create-cost`; pair with `subnet terminate-lease`
- `subnet register-neuron --netuid N` ‚Äî burn registration; unknown netuid ‚Üí exit **12** before hotkey unlock (same as `subnet show`)
- `subnet pow --netuid N [--threads 8]` ‚Äî POW registration; unknown netuid ‚Üí exit **12** before hotkey unlock; `--threads 0` ‚Üí exit **1**; no solution found ‚Üí message, exit **0**
- `subnet dissolve --netuid N` ‚Äî schedule dissolve (owner); unknown netuid ‚Üí exit **12** before wallet; confirm unless `--yes`; decline ‚Üí exit **0**
- `subnet root-dissolve --netuid N` ‚Äî root-only removal; same **`require_subnet_exists`** preflight ‚Üí exit **12** before wallet
- `subnet terminate-lease --netuid N` ‚Äî owner ends a leased subnet early; same preflight ‚Üí exit **12** before wallet
- `subnet watch --netuid N [--interval 12]` ‚Äî live tempo / rate-limit TUI (latest head each tick); unknown netuid ‚Üí exit **12** like `subnet show`; no `--output json`
- `subnet monitor --netuid N [--interval 24] [--json]` ‚Äî metagraph diff / event stream (latest head each tick); unknown netuid ‚Üí exit **12** like `subnet show`; JSON via `--json` on the subcommand (not global `--output json`)
- `subnet health --netuid N` ‚Äî miner/validator status
- `subnet emissions --netuid N` ‚Äî per-UID emissions
- `subnet cost --netuid N` ‚Äî registration cost
- `subnet liquidity [--netuid N]` ‚Äî AMM depth, slippage estimates; with `--netuid`, unknown SN ‚Üí exit **12** like `subnet show`; omit `--netuid` to scan all subnets with pool depth
- `subnet commits --netuid N [--hotkey-address SS58]` ‚Äî pending weight commits; unknown netuid ‚Üí exit **12** like `subnet show`; if CR disabled ‚Üí notice / JSON and exit **0**
- `subnet set-param --netuid N --param list|NAME [--value V]` ‚Äî list catalog (`--output json` for list) or set hyperparam (owner); unknown netuid ‚Üí exit **12** before list / wallet (same as `subnet show`)
- `subnet set-symbol --netuid N --symbol "SYM"` ‚Äî set token symbol (owner); invalid symbol ‚Üí exit **1**; unknown netuid ‚Üí exit **12** before wallet (same as `subnet set-param`)
- `subnet trim --netuid N --max-uids M` ‚Äî lower max UIDs / trim metagraph capacity (owner); unknown netuid ‚Üí exit **12** before wallet; confirm prompt unless `--yes`
- `subnet probe --netuid N [--uids "0,1,2"]` ‚Äî HTTP axon reachability (`GET /`); unknown netuid ‚Üí exit **12** like `subnet show`; pinned lite+full neuron snapshot
- `subnet check-start --netuid N` ‚Äî read-only: active flag, neuron count, `can_start`, tempo; unknown netuid ‚Üí exit **12** like `subnet show`
- `subnet start --netuid N` ‚Äî owner starts emission schedule; unknown netuid ‚Üí exit **12** before wallet (same preflight as `check-start`)
- `subnet snipe --netuid N [--fast] [--watch] [--all-hotkeys] [--max-cost TAO] [--max-attempts N]` ‚Äî block-level registration sniper; unknown netuid ‚Üí exit **12** before streaming / wallet
- `subnet cache-list|cache-load|cache-diff|cache-prune --netuid N` ‚Äî metagraph disk snapshots (`metagraph --save`); unknown netuid ‚Üí exit **12** like `subnet show`; missing cache files ‚Üí tip / empty JSON and exit **0** (see `docs/commands/subnet.md`)
- `subnet emission-split --netuid N` ‚Äî mechanism emission weights (`MechanismEmissionSplit`); unknown netuid ‚Üí exit **12** like `subnet show`; no custom split ‚Üí notice or JSON `configured: false`, exit **0**
- `subnet mechanism-count --netuid N` ‚Äî `MechanismCountCurrent` (default 1 if unset); unknown netuid ‚Üí exit **12** like `subnet show`
- `subnet set-mechanism-count --netuid N --count K` ‚Äî owner `sudo_set_mechanism_count`; unknown netuid ‚Üí exit **12** before wallet; e2e **`subnet_owner_mechanism_writes`**
- `subnet set-emission-split --netuid N --weights "a,b,..."` ‚Äî owner `sudo_set_mechanism_emission_split`; invalid weights ‚Üí exit **1** before wallet; unknown netuid ‚Üí **12** after parse; confirm unless `--yes`

### Weights
- `weights show --netuid N [--hotkey-address SS58] [--limit L]` ‚Äî read-only on-chain weights; no wallet; unknown SN ‚Üí same bail as `subnet show` (exit **12**); flaky hyperparams RPC ‚Üí warn and continue; e2e **`weights_show_preflight`** + read bundle in Phase 37 `test_all_weights`
- `weights set --netuid N --weights "uid:weight,..." [--version-key K] [--dry-run]` ‚Äî unknown SN ‚Üí same "Subnet N not found" + `subnet list` as `subnet show` (exit **12**); e2e `weights_set_preflight` + Phase 7 `set_weights`; `--dry-run` still unlocks wallet for stake check; on-chain `WeightVecLengthIsLow` when vector shorter than `min_allowed_weights` (see `subnet hyperparams`)
- `weights commit --netuid N --weights "uid:weight,..." [--salt S]` ‚Äî unknown SN ‚Üí same bail as `subnet show` (exit **12**) before wallet; e2e **`weights_commit_preflight`** + Phase 17 `commit_weights` (no pre-submit stake check unlike `weights set`)
- `weights reveal --netuid N --weights "uid:weight,..." --salt S` ‚Äî same unknown-SN pre-check as `commit`; e2e **`weights_reveal_preflight`** + Phase 17 `reveal_weights` (`test_reveal_weights_rejected_without_prior_commit` and follow-on reveal cases)
- `weights commit-reveal --netuid N --weights "uid:weight,..." [--version-key K] [--wait]` ‚Äî atomic commit+wait+reveal (or `set_weights` when CR off); unknown SN ‚Üí exit **12** before wallet; hyperparams RPC failure aborts (unlike commit/reveal/status); e2e **`weights_commit_reveal_preflight`** in Phase 17 `test_commit_weights_rejected_when_commit_reveal_disabled`
- `weights status --netuid N` ‚Äî pending commit-reveal rows for default hotkey (`get_weight_commits` + head + hyperparams + `reveal_period_epochs`); unknown SN ‚Üí same hyperparams preflight (exit **12**) before wallet; RPC warn+continue on preflight; e2e **`weights_status_preflight`** in Phase 17 `test_reveal_weights_rejected_without_prior_commit`
- `weights commit-timelocked --netuid N --weights "..." --round R` ‚Äî drand timelock commit; unknown SN ‚Üí exit **12** before wallet (same hyperparams preflight as `weights commit`; RPC warn+continue); SDK reads `CommitRevealWeightsVersion` at submit; e2e **`weights_commit_timelocked_preflight`** in Phase 17 `test_commit_timelocked_weights_rejected_when_incorrect_commit_reveal_version`; on-chain **111** = version mismatch (rare with current CLI)
- `weights set-mechanism --netuid N --mechanism-id M --weights "..." [--version-key K]` ‚Äî `set_mechanism_weights`; unknown SN ‚Üí exit **12** before wallet (`require_subnet_exists_for_weights_cmd`; RPC warn+continue); **`--dry-run`** = JSON only, no wallet; e2e **`weights_set_mechanism_preflight`** in Phase 5 `test_set_mechanism_weights`
- `weights commit-mechanism --netuid N --mechanism-id M --hash H` ‚Äî `commit_mechanism_weights`; same preflight as `set-mechanism`; **`H`** = 32-byte hex (blake2 over uids+weights+salt like `weights commit`); invalid length/hex ‚Üí exit **1** before RPC; e2e **`weights_commit_mechanism_preflight`** in Phase 5 `test_commit_mechanism_weights`
- `weights reveal-mechanism --netuid N --mechanism-id M --weights "..." --salt S [--version-key K]` ‚Äî `reveal_mechanism_weights`; same preflight as `set-mechanism` / `commit-mechanism`; **`S`** ‚Üí little-endian u16 pairs like **`weights reveal`**; bad **`--weights`** ‚Üí exit **1** before RPC; e2e **`weights_reveal_mechanism_preflight`** in Phase 5 `test_reveal_mechanism_weights`

### Block
- `block latest` ‚Äî read-only head (no wallet); `get_block_number` ‚Üí `get_block_hash` ‚Üí extrinsic count + timestamp; e2e **`block_latest_preflight`** in Phase 20 `test_block_queries`; network **10** / timeout **15** / generic **1**; head \> `u32::MAX` ‚Üí **1** (theoretical)
- `block info --number N` ‚Äî `get_block_hash` ‚Üí header + extrinsic count + timestamp; e2e **`block_info_preflight`** in Phase 20 `test_block_queries`; same exit classification as `block latest`
